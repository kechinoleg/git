Задание 1:  Как скопировать на свой компьютер репозиторий из github.com, 
что должно быть сделано предварительно?
Выполнение:
1.Перейти в директорию в которой нужно сохранить репозиторий (cd/полный/ 
путь/)
2.Выполнить команду (git clone/ссылка на репозиторий/)

Задание 2: Как создать git репозиторий из локального каталога на своем 
компьютере. Как его отправить на github.com?
1. Создать директорию, в ней можно создать файл или несколько
2. Выполнить команду git init
3. Выполнить команду git add . или git add <название_файла> избирательно
4. Выполнить команду git commit -m 'описание изменений'
5. Выполнить команду git push

Задание 3: Как получить на локальный компьютер изменения с github.com
1. Войти в нужную директорию и выполнить команду git pull

Задание 4: Какую информацию нам дают команда :
git status показывает изменения файлов.
git log выводит историю коммитов.
git remote -v выводит список удалённых репозиториев.
